process BASE

2021-08-16 Node JS, Node 0 评论

node –base!

这篇内容比较杂

  • process.argv

    process.argv -> 以数组的方式显示命令行内容

  • process.exit()

    process.exit() -> 强制退出程序

  • 命令行 node -p -e 'proces.env' 可以输出电脑中node的环境变量

    接下来这个有意思了！！！ 注意听 同学们！

  • process.env.npm_package_

    process.env.npm_package_ -> 可以取的package.json里面定义好的key:value;首先，通过npm_package_前缀，npm 脚本可以拿到package.json里面的字段。比如，下面是一个package.json。

    npm run view

那么，变量npm_package_name返回foo，变量npm_package_version返回1.2.5。

  • ts-node

    ts-node -> 直接启动ts项目
    用法： node -r ts-node/register script/build --env=development --watch script/build.ts!!!

  • rollup-plugin-esbuild

    rollup-plugin-esbuild -> 直接去 esbuild md 粘贴！！！// ts 编译为js

  • chalk

    chalk -> 输出代码加颜色！

  • ora

    ora -> npm 依赖包
